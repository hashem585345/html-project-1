# 🌐 HTML Learning Project

This project is a comprehensive HTML practice project created to learn and demonstrate core HTML concepts through a single structured webpage.

---

## 📌 Project Overview

This project contains examples of almost all fundamental HTML elements, including:

- Headings (h1 - h6)
- Paragraphs and text formatting
- Images and figures
- Links and navigation
- Lists (ordered, unordered, description lists)
- Tables with rowspan and colspan
- Forms with different input types
- Multimedia (audio and video)
- iframe embedding
- Semantic and structural elements like div, span, blockquote, details, and more

---

## 🧠 What You Will Learn

By exploring this project, you will understand:

- How to structure an HTML page properly
- How to use text formatting tags
- How to insert and manage images and videos
- How to create and style links
- How to build tables with advanced features
- How to create interactive forms
- How to embed external content using iframe
- How to organize content using semantic HTML elements

---

## 📁 Project Structure

- `h.html` → Main HTML file containing all examples
- `fotos/` → Folder containing images
- `video/` → Folder containing videos and audio files

---

## 🎯 Purpose

This project is designed for:
- Beginners learning HTML
- Practicing real-world HTML structure
- Understanding how different HTML elements work together
